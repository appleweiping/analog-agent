# Cold Knowledge

This directory contains compact, durable notes used by the agent harness. Keep
it small and curated; generated paper drafts, raw logs, and experiment outputs
belong under `archive/`.

Suggested contents:

- `analog_rules/`: stable design heuristics and physical constraints.
- `equation_cards/`: formulas with units and known validity boundaries.
- `paper_snippets/`: short related-work notes and positioning reminders.
- `topology_notes/`: topology families, design variables, and failure modes.

Do not treat this as a dumping ground. Every file should help a future agent
plan, verify, or explain a concrete analog-design task.
